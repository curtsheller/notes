# Common Progression

In Rock, there is no more common progression than “The Blues”. Start with E Blues.

## Basic Blues
```
| E / / / | / / / / | / / / / | / / / / |
| A / / / | / / / / | E / / / | / / / / |
| B / / / | / / / / | E / / / | / / / / |
```

## Basic Blues with V Turnaround
This version adds the IV chord in bar 10 and the V chord in bar 12.

```
| E / / / | / / / / | / / / / | / / / / |
| A / / / | / / / / | E / / / | / / / / |
| B / / / | A / / / | E / / / | B / / / |
```

## Quick Four
Basic Blues with V Turnaround
This version adds the IV chord in bar 10 and the V chord in bar 12.

```
| E / / / | A / / / | E / / / | / / / / |
| A / / / | / / / / | E / / / | / / / / |
| B / / / | A / / / | E / A / | E / B / |

```
