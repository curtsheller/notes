# Images

## webp Support

Add **WebP** support to GNOME Image Viewer in Ubuntu and Other Linux.

```sh
sudo add-apt-repository ppa:helkaluin/webp-pixbuf-loader
sudo apt update
sudo apt install webp-pixbuf-loader
```
